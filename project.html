CV Generator

Project Description

The CV generator is a project that I worked upon when I was in the first year of my Bachelors Degree. It was one of my Pioneer project. We prepared a program that generates a resume when we enter certain details of ours. Who use programming to achieve this and ensured that the project will beneficial for the beginners as well as the intermediate who are struggling to generate decent and appropriate resume for their profile.

In the initial stages of our programming journey, we aimed to create a foundational program that outputs a well-structured and organized text-based CV format. As we delved deeper into the project, we recognized the potential for enhancement. This realization led us to develop a comprehensive application capable of generating a complete CV format. Our program not only produces a polished textual template but also incorporates visual elements, such as images, culminating in a refined and aesthetically pleasing CV for users.

The implementation was executed in C++. Throughout the development process, we encountered several challenges, particularly in managing external libraries. We faced difficulties with installation on Windows, which sometimes resulted in program termination due to an inadequate environment for code execution. In contrast, transitioning to a Linux environment significantly streamlined our workflow. The simplicity of installing external libraries in Linux—with just a single command—greatly facilitated our development efforts.
